---

####INSTALLATION
Copy the folder *RotateIndividuals* to *Scripts/Scripts Panel* of your InDesign installation.

####HOW TO USE
In InDesign, select the object you want to rotate.
Open the scripts window and double click *RotateIndividuals.jsx*.
